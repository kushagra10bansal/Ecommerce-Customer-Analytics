# E-Commerce Customer Analytics & RFM Segmentation

## Project Overview

This project analyzes customer purchasing behavior using the Olist Brazilian E-Commerce Dataset.

The goal is to identify high-value customers, understand revenue trends, analyze product performance, and perform customer segmentation using RFM Analysis.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SQLAlchemy
* SQLite
* OpenPyXL

## Business Questions Answered

* What is the total revenue?
* What are the monthly revenue trends?
* Who are the top customers?
* Which product categories generate the most revenue?
* Which states contribute the most revenue?
* What customer segments exist based on RFM analysis?

## Key Features

### KPI Dashboard

* Total Revenue
* Total Customers
* Total Orders
* Average Order Value

### Customer Analytics

* Top Customers
* Repeat Customer Analysis
* RFM Segmentation

### Product Analytics

* Top Product Categories
* Payment Method Distribution

### Database Integration

The cleaned dataset is stored in SQLite using SQLAlchemy and queried using SQL.

### Excel Reporting

Customer segmentation results are exported automatically to Excel using OpenPyXL.

## Dataset

Olist Brazilian E-Commerce Public Dataset (Kaggle)
